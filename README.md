## Como instalar e executar
Clone o repositório:
git clone LINK_DO_REPOSITORIO
Acesse a pasta do projeto:
cd NOME_DA_PASTA
Instale as dependências:
npm install
Execute o projeto:
npm run dev
Caso tenha backend:
cd backend
npm install
npm start
