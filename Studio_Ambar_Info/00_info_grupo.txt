STUDIO ÂMBAR
============================================================

Funções sugeridas:
- Liderança do grupo: organização do GitHub, cronograma e integração geral.
- Front-end React: desenvolvimento de páginas e componentes.
- UI/UX: layout, responsividade e identidade visual.
- Back-end/API: banco de dados MySQL e área administrativa.

Empresa fictícia:
Studio Âmbar

Área de atuação:
Estúdio criativo de design, branding e conteúdo digital.

Histórico da empresa:
A Studio Âmbar é uma empresa fictícia especializada em identidade visual, criação de marcas, design para redes sociais e soluções digitais para pequenos negócios. A empresa busca transformar ideias em marcas fortes, com comunicação visual profissional e presença digital organizada.

Objetivo do site:
Criar um site de portfólio e serviços para divulgar os trabalhos do estúdio, apresentar cases, publicar conteúdos e receber contatos de potenciais clientes.

Páginas esperadas:
- Home
- Sobre
- Serviços
- Portfólio
- Cases
- Blog
- Contato
- Área Admin

Funcionalidades obrigatórias:
- Listagem dinâmica de serviços.
- Portfólio com projetos/cases.
- Blog ou área de publicações.
- Formulário de contato.
- Área administrativa para gerenciar serviços, posts e projetos.
- Integração com banco MySQL.

Banco de dados esperado:
- clientes
- servicos
- projetos
- portfolio
- posts
- contatos
- usuarios

Etapas do projeto:
Etapa 1 — Identidade visual e páginas iniciais:
Criar layout base, componentes, páginas institucionais e estrutura de navegação.

Etapa 2 — Integração React + API + MySQL:
Transformar serviços, portfólio e posts em dados dinâmicos vindos da API.

Etapa 3 — Portfólio dinâmico, painel e entrega final:
Criar administração dos conteúdos, finalizar responsividade, testar e apresentar o site.
